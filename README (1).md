# Sky.Cast 🌤️
### Smart Weather Intelligence for Every Nigerian

A real-time, AI-augmented weather web application built for the **NextGen by Airtel & 3MTT Knowledge Showcase 2025**.

---

## 🌍 Live Demo
> [skycast-ng.netlify.app](https://skycast-ng.netlify.app)

---

## 📌 What It Does

SkyCast transforms raw weather data into rich, visual, human-readable forecasts — paired with AI-generated advice tailored to your **profession** and **conditions**.

Most weather apps tell you it's raining. SkyCast tells you what to do about it.

---

## ✨ Features

- **📍 Auto-detect location** — one click, instant weather for where you are
- **🤖 AI advice engine** — dynamic tips that change based on temperature, humidity, wind, and condition
- **👤 Profession-based advice** — select Farmer, Student, Commuter, Driver, Athlete and more — advice adapts to your life
- **🌧️ Dynamic hero backgrounds** — real sky photos that change with the weather and time of day
- **📸 Pictorial forecast cards** — city cards, daily cards, and hourly cards all show sky photos with overlaid data
- **📅 Daily forecast** — search any city, pick tomorrow / next 2–3 days
- **🕐 Hourly forecast** — next 12 hours with temperature chart and per-hour cards
- **🌍 Popular cities** — Lagos, Abuja, London, New York, Ado-Ekiti, Ibadan loaded live on page load
- **🧠 Smart city aliases** — type "Ekiti", "FCT", "Adebayo" and it resolves correctly
- **📱 Fully responsive** — mobile-first, works on any screen or browser

## 🚀 Run Locally

No setup needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/yourusername/skycast.git

# Open in browser
open skycast-v2.html
```

Or simply download `skycast-v2.html` and open it in any browser.

## 🔑 API Key

This project uses the [OpenWeatherMap API](https://openweathermap.org/api) (free tier).
